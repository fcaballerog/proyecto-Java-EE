Instrucciones para ejecutar el proyecto:
 - Crear en MySQL una base de datos con el nombre mtd, Importar el archivo sql adjunto en la carpeta Db/mtd.sql que esta en el proyecto.
 - Importar proyecto al espacio de trabajo de eclipse.
 - Lanzar xamp arrancando apache y mySQL.
 - Editar el archivo hibernate.cfg.xml acorde con los datos de acceso de
	phpMyAdmin (y variar el nombre de la BD en caso de que se haya
	creado una nueva con distinto nombre).
 - Comprobar que phpMyAdmin está ejecutandose y lanzar el proyecto desde el servlet es.fcg.principal.Principal.java.
